# 09 · Patterns

## Ключевые концепции

- **Compound Components** — неявный state через Context, гибкая композиция
- **Render Props** — переиспользование логики с кастомным UI
- **HOC** — оборачивание компонента, `displayName`, композиция
- HOC vs Custom Hooks: когда что использовать
