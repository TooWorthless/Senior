HTML Security: примеры и антипаттерны

✅ Внешние ссылки: rel="noopener noreferrer"

Безопасная внешняя ссылка

window.opener === null, Referer не отправляется

❌ Уязвимая ссылка (reverse tabnapping)

<a href="..." target="_blank"> без rel="noopener"

Открытая вкладка имеет доступ к window.opener

✅ iframe с sandbox

sandbox="allow-scripts" — только JS, без form submit, без same-origin

✅ Безопасная вставка данных через textContent

❌ Опасная вставка через innerHTML (XSS)

Никогда не делайте: element.innerHTML = userInput

✅ Форма с CSRF защитой

✅ Валидация href перед вставкой

✅ crossorigin на скриптах для stack trace

<script src="https://cdn.example.com/app.js" crossorigin="anonymous">

Без crossorigin: window.onerror получает только "Script error."
