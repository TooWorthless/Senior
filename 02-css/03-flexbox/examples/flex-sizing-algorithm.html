Demo 3: flex-grow:1 (basis:auto) vs flex:1 (basis:0)

flex-grow: 1 (auto basis) — неравные колонки:

A — длинный текст контент
B — средний
C

flex: 1 (basis:0) — равные колонки:

A — длинный текст контент
B — средний
C

flex: 1 = flex: 1 1 0 — все базисы 0, пространство делится поровну.
flex-grow: 1 при flex-basis: auto — базис = контент, пространство делится поверх него.

Demo 4: min-width: 0 — разрешает элементу сжаться

❌ Без min-width:0 — этот длинный текст ломает layout и не обрезается как надо
✅ С min-width:0 — этот длинный текст корректно обрезается с ellipsis

Flex items имеют min-width: auto по умолчанию. Это значит они не сожмутся меньше min-content. min-width: 0 снимает это ограничение.
